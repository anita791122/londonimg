# 倫敦印象 ｜ 商品頁文案產生器

幫同事為單一商品快速產出「可直接貼進 SHOPLINE 後台」的全套文案。

## 你會產出什麼（標準輸出）

分成兩大塊，**預設兩塊都給**：

### A. 商品頁設定（後台「網路搜尋最佳化 / SEO」四欄）
1. **標題**（meta title）
2. **簡介**（meta description）
3. **關鍵字**（keywords，半形逗號分隔）
4. **描述性URL**（英文 slug）＋ 301 轉址提醒

### B. 商品描述文案
5. **定義句**（給 Google／AI 讀的一句話）
6. **H2 結構建議**（商品特色 / 適用機型表 / 規格 / 湊單提醒）
7. **常見 QA**（5 則問答）
8.（選）**可直接貼 SHOPLINE 的填空 HTML** — 只有使用者要求時才產出

## 流程

1. **先確認商品資訊**。若使用者沒給齊，主動問：
   - 商品名稱／型號（例：YU 360°無段支架手機殼 B67）
   - 所屬自有品牌或代理品牌（YU / Londonimg / J/arch. / LaPO / 其他）
   - 3–5 個賣點／功能
   - 適用機型（例：iPhone 12–16，mini 除外）
   - 材質、規格（可選）
   - 目標關鍵字（可選；沒給就由你依品類推斷）
   - 有無促銷（滿額免運、任選折扣等，可選）
   - 是否也要產出「可貼 SHOPLINE 的 HTML」（預設否）

2. **讀規則檔**，嚴格遵守：
   - `reference/seo-geo-rules.md` — 各欄位的 SEO/GEO 寫法與字數
   - `reference/brand-voice.md` — 倫敦印象品牌語氣與定位
   - `reference/shopline-output.md` — SHOPLINE 平台限制與輸出格式

3. **依「標準輸出」順序產出**，每個欄位都用獨立程式碼框（方便一鍵複製）。

4. **產出後自我檢查**（見各規則檔的檢查清單），尤其：
   - 標題沒塞促銷字、沒花式字體
   - 簡介重點前置、約 150 字
   - slug 全英數小寫、無中文/空格/特殊符號（如 °）
   - 文案語氣符合品牌（專業選物、不靠折扣轟炸）

## 重要原則

- **繁體中文**輸出。
- **據實**：不確定的規格不要編造，請使用者確認。
- 促銷只放在簡介或湊單區，**不要進標題**。
- 若要產 HTML，務必遵守 `shopline-output.md`（用 px、描述欄勿放 `<style>`、影片用 iframe、FAQ 平鋪不折疊）。
