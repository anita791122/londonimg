# SHOPLINE 平台限制與輸出格式

> 產出文案／HTML 時務必遵守，否則貼進後台會跑版或被濾掉。（皆為實測結論）

## 後台貼放位置
- **SEO 四欄**：商品 → 編輯 → **網路搜尋最佳化**。標題／簡介／關鍵字／描述性URL 各一欄。
- **商品描述**：商品 → 商品描述 → 右上角 `</>` 切「HTML 原始碼」模式再貼。

## 商品描述欄位會「淨化」HTML（重要）
- `<style>`、`<script>` → **整段被移除**（CSS/JS 手風琴會失效、攤平）。
- `<details>` / `<summary>` → **連標籤都被濾掉**，折疊失效。
  → 所以 **FAQ 要做「卡片式平鋪」**（全展開也好看，且全文可見利 SEO），不要做折疊。
- `<iframe>`（影片）→ **保留**。用 YouTube `embed/` 直式 iframe，**不要**用 `aspect-ratio` 比例盒包它（會露黑邊）。

## 單位一律用 px，不要用 rem
- SHOPLINE 主題設了 `html{font-size:62.5%}`，貼進來的 `rem` 會被縮成 62.5%（圖變小、字變小）。
- 響應式改用 `clamp()` / `vw` / `%` / `aspect-ratio`（不受根字級影響）；固定尺寸用 px。

## 若要產出「可貼 HTML」（使用者要求時）
- ⚠️ **一律分段給，不要給一整包**（與教學頁一致）：**每個區塊各自一個獨立程式碼框**，讓同事像教學頁那樣「需要哪塊就複製哪塊」、不需要的略過。**絕不可**把全部區塊塞進同一個 `<div>` 一次給。
- **每個區塊框的開頭標出區塊名稱**（用教學頁同名：首屏 Hero／定義句／特色區①…／對照表／版本卡片／使用情境／功能特色／適用機型／規格／常見 QA），例如：
  - `【可貼 HTML — 首屏 Hero】` ＋ 其下一個 ```html 框
  - `【可貼 HTML — 特色區①】` ＋ 其下一個 ```html 框
- **每塊都要能單獨貼、單獨成立**：每塊是一個自成一體的 `<div style="…">…</div>`，同事單獨複製貼進商品描述就能正常顯示，不依賴其他塊。
- **每塊一律不設 max-width（用滿版/100%）（重要）**：經實測（含 2400px 寬螢幕），SHOPLINE **商品描述欄會被「頁面版位」自動限寬**在約 **1300–1400px**（外層 `block-inner` 1400px、描述欄 `.ProductDetail-description` 內容約 1310px）並**置中**——這個限寬是 SHOPLINE 頁面做的，不是描述內容自己。所以每塊**都不要再加 `max-width`**、也不要外包限寬層：滿版（100%）的區塊會剛好填滿那個 ~1310px 描述欄、與官網一致。**若自己加 max-width（如 900／1000）反而會比官網描述欄還窄、縮一截**。左右用小 padding（如 `padding:34px 4px`）留白即可；適用機型／規格內層若有 `max-width` 也一併**拿掉**。
- **按黃金結構順序由上到下給**：影音實測 → 首屏 Hero → 定義句 → 特色區①②③ → 對照表 →版本卡片 → 特色區④ → 使用情境 → 功能特色 → 適用機型 → 規格 → 常見 QA。同事依序往下貼即是正確順序；要跳過某塊直接不複製即可。
- **全 inline style**（每個元素 `style="…"`），不要外部 CSS／不要 `<style>`。
- 響應式靠 `clamp()`＋`flex-wrap`＋`aspect-ratio` 內嵌，**不依賴 @media**（這樣即使被濾 style 也還在）。
- **圖片網址佔位文字要跟教學頁逐字相同**：每個 `<img>` 的 `src` 都寫成 `src="✏️【請貼上圖片網址（見「圖片取得方式」分頁）】"`（與教學頁的圖片佔位**完全一致**，不要簡寫、不要加序號）。
  - 同事流程：先把圖上傳 SHOPLINE 取得 `https://img.shoplineapp.com/...`，再搜尋 ✏️ 把每個佔位換成網址；同一區塊多張圖照「上到下／左到右」順序對應即可。
- 每塊內沿用 `.yu-pdp` 黃金結構的對應段落樣式。

## 輸出呈現（給同事複製）
- 每個欄位／區塊用**獨立程式碼框**，方便一鍵複製、按需挑用。
- SEO 四欄各自一框；商品描述**逐區塊各一框**（不論是純文字版或可貼 HTML 版，都是一塊一框，不要整包）。
- 純文字版：每塊用 `【區塊：◯◯】` 當標題列欄位文字。可貼 HTML 版：每塊用 `【可貼 HTML — ◯◯】` 當標題，其下一個 ```html 框。

## 文案產出後提醒使用者
- 改了 slug → 去設 **301 轉址**。
- 圖片網址要先上傳 SHOPLINE 再複製；貼完 HTML 後，**搜尋 ✏️ 把每個 `✏️【請貼上圖片網址（見「圖片取得方式」分頁）】` 都換成真實圖片網址**（還留著 ✏️ 代表那張圖還沒換）。
- 描述欄貼 HTML 前先切「HTML 原始碼」模式。
