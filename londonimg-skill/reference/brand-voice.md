# 倫敦印象 品牌語氣與定位

## 品牌背景
- 網站：londonimg.tw（SHOPLINE 平台）
- 業態：**iPhone 配件集合店**（授權經銷商，非品牌方）
- 自有品牌：**YU、Londonimg、J/arch.、LaPO**
- 也代理多個外部配件品牌（嚴選策展）

## 客群（三型）
1. **目標型**：已知機型，要快速找到相容配件 → 文案要明確標機型、相容性。
2. **撿便宜型**：先看優惠 → 可帶免運/湊單，但不靠它當主訴求。
3. **探索型**：還不知道買什麼 → 用情境、好處、使用畫面引導。

## 語氣原則
- **繁體中文**、口語但專業，像「懂 3C 的選物店員」。
- **專業選物感 > 促銷轟炸**：定位是嚴選策展，不要通篇折扣。
- 講**好處與使用情境**，不要只列規格（規格放規格區）。
- 自有品牌（YU 等）可帶一點設計／質感敘事；代理品牌據實描述。
- 避免浮誇、避免絕對化用語（最便宜、第一）；避免花式字體與過量 emoji。

## 差異化定位（重要）
- **不過度依賴折扣**（會稀釋專業選物形象）。促銷只當輔助、放簡介或湊單區。
- 強調：相容性清楚、選物有觀點、配件齊全（殼/貼/MagSafe/鏡頭環/掛繩…）。

## 寫文案時
- 機型相容性要**講清楚**（含 Pro/Pro Max/Plus，標明 mini 是否適用）。
- 有自有品牌時，品牌名放標題前段強化品牌資產。
- 賣點轉成「對使用者的好處」（例：360°支架 → 追劇/視訊免手扶）。
